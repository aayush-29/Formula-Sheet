# 📚 Chapter Wise Formula Sheet

A Python project to display Physics & Mathematics formula sheets as images!

## 🧠 What I Learned Here:
- 📷 Load & display images using **PIL/Pillow**
- 📂 Handle file paths in Python
- ⚙️ Build interactive menu systems
- 🎨 Work with visual content programmatically

## 🚀 Quick Start:
```bash
pip install pillow
python formula_sheet.py
```

motive of this project was to learn how to compile image in python using external library :D
