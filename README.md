# 📚 Chapter Wise Formula Sheet 

## 🎯 Project Overview

A simple Python application that displays **Physics and Mathematics formulas** from image files based on user selection. This project was created to learn how to **work with images in Python** using the `PIL (Pillow)` library.

---

## 📖 What Does It Do?

✨ The application allows students to:
- Select a subject (**Physics** or **Mathematics**)
- Choose a chapter from the available list
- View formula sheets as **images** directly in the Python environment

---

## 🧠 Learning Objective

> **Main Motive:** This project teaches how to:
- 📷 **Load and display images** in Python using PIL/Pillow
- 📂 **Manage file paths** and handle image files
- 🎨 **Work with visual content** programmatically
- ⚙️ **Build a simple interactive menu system**

---

## 📦 Installation

### Prerequisites:
```bash
pip install pillow
```

### How to Use:
```bash
python formula_sheet.py
```

1. Enter the subject name (Physics/Mathematics)
2. Select a chapter number from the list
3. View the formula image! 📸
